# 專案更新歷史 (Update History)

## [2026-04-02]
### 新增功能
- **音樂播放功能**：在專注倒數頁面底部新增音樂播放選項。
    - 支援兩種匯入方式：「選擇音樂檔案」與「匯入音樂資料夾」。
    - 支援清空目前的音樂清單。
    - **專注期間選歌**：在專注開始後，原先的「選擇音樂」按鈕會變成「選擇播放音樂」，點擊後可直接從現有清單中挑選當下想播放的曲目，且會即刻播放（若原本設為隨機播放，則會關閉隨機並從指定歌曲繼續）。修復了選歌後無法立刻成功播放的問題。
    - 加入防呆機制：若選擇的音樂檔案已從手機中被刪除，系統會自動略過該檔案並嘗試播放下一首，避免程式崩潰。
    - 音樂清單會永久儲存在裝置內，重啟程式不會遺失資料。
- **倒數結束鬧鐘功能**：
    - 在專注狀態文字下方新增「開啟鬧鐘」選項。
    - 開啟後，倒數結束時會自動發出「復古機械雙鈴鬧鐘」的專屬音效提醒。
    - 鬧鐘可手動關閉，或在 5 分鐘後自動停止。
- **UI 佈局優化**：
    - 修復了主畫面底部溢出 (Bottom Overflow) 的問題。
    - 加入 `SingleChildScrollView` 支援滾動。
    - 優化了元件之間的間距與 Padding，使介面在不同螢幕尺寸上更協調。
- **Bug 修正與程式碼優化**：
    - 修復音樂播放完成事件監聽器重複註冊導致記憶體洩漏的問題。
    - 修復 `dispose()` 未釋放 AudioPlayer 資源的問題。
    - 修復鬧鐘觸發後 UI 可能一閃而過的事件順序問題。
    - 加入遞迴保護，防止所有音樂檔案失效時的無限迴圈。
    - 修復番茄鐘休息模式下音樂不會自動繼續播放的問題。
    - `musicPaths` getter 改為回傳不可變列表，防止外部意外修改。

### 環境與架構
- **新增 Windows 平台支援**：執行 `flutter create --platforms=windows .` 以支援 Windows 桌面預覽。
- **新增依賴套件**：
    - `file_picker`: 用於選取音樂檔案。
    - `audioplayers`: 用於播放專注音樂與結束鬧鈴。
- **更新多語言系統**：新增音樂與鬧鐘相關的繁體中文、英文與日文翻譯。
- **專案結構更新**：更新 `PROJECT_STRUCTURE.md` 以反映新增的平台與功能模組。
