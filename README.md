# 3c_detox

## 中文

`3c_detox` 是一個 Flutter Android 專案，主題是數位排毒與專注管理。App 名稱目前是 `FocusFlow`，核心做法是在專注期間封鎖高干擾 App，降低分心。

### 核心功能
- 黑名單 App 管理
- 專注模式
- 番茄鐘模式（Pomodoro）
- 使用狀況權限與懸浮權限配合攔截
- 專注統計與進度查看
- 本地偏好的儲存
- 音效播放

### 主要技術
- Flutter / Dart
- `provider`
- `shared_preferences`
- `file_picker`
- `audioplayers`
- `google_fonts`

### 專案狀態
目前 repo 內已經有實際 App 功能與操作說明，不再只是 Flutter 模板。這是一個明確以 Android 專注管理為目標的產品原型。

## English

`3c_detox` is a Flutter Android project for digital detox and focus management. The current app name is `FocusFlow`, and its core mechanism is blocking distracting apps during focus sessions.

### Core features
- blocked-app list management
- focus mode
- Pomodoro mode
- interception flow built around usage-access and overlay permissions
- focus statistics and progress tracking
- local preference storage
- audio playback

### Main technologies
- Flutter / Dart
- `provider`
- `shared_preferences`
- `file_picker`
- `audioplayers`
- `google_fonts`

### Project status
This repository already contains a real application direction and operating instructions. It is no longer just a default Flutter scaffold; it is a concrete Android focus-management product prototype.
