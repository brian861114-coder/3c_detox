# FocusFlow 使用說明手冊

FocusFlow 是一款極簡主義（Zen style）的 3C Detox 行動應用程式，旨在幫助您在特定時間內專注，並透過封鎖特定的應用程式（如社交媒體或遊戲）來減少分心。

## 📦 如何安裝到手機上測試

要將 FocusFlow 安裝到您的 Android 實體手機上，請按照以下步驟操作：

### 1. 開啟手機的「開發者模式」與「USB 偵錯」
1. 打開手機的 **「設定」 (Settings)**。
2. 滑動到最下方，點擊 **「關於手機」 (About Phone)**。
3. 找到 **「版本號碼」 (Build Number)** （某些廠牌如小米是在「MIUI 版本」，三星在「軟體資訊」內）。
4. **連續點擊「版本號碼」 7 次**，直到畫面顯示「您現在已成為開發人員！」。
5. 回到上上一層的「設定」選單，找到並進入 **「開發者選項」 (Developer Options)**（有時會藏在「系統」或「其他設定」裡面）。
6. 在「開發者選項」中，找到 **「USB 偵錯」 (USB Debugging)** 並將其 **開啟**。

### 2. 連接手機並允許存取
1. 使用 USB 傳輸線將手機連接到電腦。
2. 手機螢幕上可能會彈出提示：**「允許 USB 偵錯嗎？」(Allow USB debugging?)**
3. 勾選「一律允許透過這台電腦進行」，然後點擊 **「允許」**。

### 3. 編譯並安裝 App
在您的電腦終端機 (Terminal) 中，確認終端機路徑位於 `focus_flow` 專案資料夾底下，然後輸入：
```bash
flutter run
```
系統會進行編譯，完成後 FocusFlow App 就會自動安裝並在您的手機上開啟了！

## ⚙️ App 功能設定與操作指南

### 1. 初始權限設定 (非常重要！)
為了讓 FocusFlow 能夠成功攔截您想封鎖的 App，您必須先賦予它原生系統權限。
1. 打開 FocusFlow，點擊主畫面右上角的 **清單圖示 (App Selector)**。
2. 在「Blocked Apps」頁面中，點擊右上角的 **齒輪圖示 (Settings)**。
3. 系統會引導您到手機設定，請手動找到 FocusFlow 並開啟以下兩個權限：
   - **使用狀況存取權 (Usage Access)**：用來偵測目前畫面上開啟的是哪個 App。
   - **顯示於上方 / 懸浮視窗 (Display Over Other Apps)**：用來在您偷開黑名單 App 時，強制將畫面跳轉回 FocusFlow。

### 2. 設定黑名單 (封鎖 App)
1. 點擊主畫面右上角的 **清單圖示** 進入「Blocked Apps」頁面。
2. 畫面會列出您手機上安裝的所有應用程式。
3. 找到您想在專注時間內封鎖的 App（例如：Instagram, YouTube, 遊戲等），將右側的開關 **打開**。

### 3. 開始專注與番茄鐘模式 (Pomodoro)
1. 回到首頁，畫面上方有一個 **「Pomodoro Mode」 (番茄鐘模式)** 的開關。
   - **開啟 (預設)**：採用經典番茄鐘作息（25 分鐘專注 ➡️ 5 分鐘休息）。休息時間內，黑名單攔截會暫時解除。
   - **關閉**：單次 30 分鐘的專注挑戰。
2. 點擊畫面中央巨大的 **「START FOCUS」** 按鈕。
3. 倒數計時開始，螢幕會顯示「Focus Mode Active」且背景啟動嚴格模式！

### 4. 嚴格模式測試
1. 在倒數計時期間（非 Resting 休息狀態），按下手機的 Home 鍵回到手機桌面。
2. 嘗試點開您剛剛加入黑名單的 App。
3. FocusFlow 的常駐防護機制會瞬間偵測到，並將您 **強制拉回 FocusFlow 的專注畫面**，防止您繼續滑手機！

### 5. 查看專注統計數據
1. 點擊主畫面右上角的 **長條圖圖示 (Stats)**。
2. 在「Your Progress」畫面中，您可以查看：
   - **Pomodoros Completed**：您今天成功完成的番茄鐘循環次數。
   - **Total Focus Time**：您累積的總專注時間（分鐘）。
